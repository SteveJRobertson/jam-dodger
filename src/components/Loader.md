### Loader examples

#### Default

```js
<Loader />
```

#### Examples with props

The size of the loader can be adjusted by using the Semantic UI loader size clases. The default
class is 'massive'. Text is optional.

---

```js
<Loader
  size={'mini'}
  text={'Mini loader'}
/>
```

```js
<Loader
  size={'tiny'}
  text={'Tiny loader'}
/>
```

```js
<Loader
  size={'small'}
  text={'Small loader'}
/>
```

```js
<Loader
  size={'medium'}
  text={'Medium loader'}
/>
```

```js
<Loader
  size={'large'}
  text={'Large loader'}
/>
```

```js
<Loader
  size={'massive'}
  text={'Massive loader'}
/>
```
